# Hangman Game 🎮

A simple text-based Hangman game developed using Python.  
The player tries to guess a hidden word one letter at a time within limited chances.

---

## 📌 Project Description

This project is a beginner-friendly Python game that demonstrates:

- Random word selection
- Loops and conditional statements
- String handling
- Lists
- User input/output

The game selects a random word from a predefined list and allows the player to guess letters.  
The player wins if the complete word is guessed before running out of chances.

---

## 🚀 Features

- Simple console-based interface
- Random word generation
- 6 incorrect guess limit
- Input validation
- Win/Lose conditions
- Beginner-friendly Python project

## 🛠 Technologies Used
